Bem-vindos a aula de Git

lista de comando

  • git status: mostra status do projeto no git
  • git add [file] : adiciona o arquivo (ou mudanças) no controle do git
  • git rm --cached [file]: remove o arquivo (ou alterações) do controle do git
  • git commit: cria um novo commit na linha do tempo do projeto com as alterações da staging area
  • git push: envia os commits locais para o repositorio remoto
